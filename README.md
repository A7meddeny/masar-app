# تطبيق مسار - المناديب (Android)

تطبيق أندرويد يربط المناديب ببوابة الخدمات في موقعك.

## كيف تحصل على APK جاهز للتثبيت (بدون تثبيت أي شيء)

### الخيار 1 — عبر GitHub Actions (مجاني، يأخذ 5-7 دقائق)

1. **أنشئ حساب GitHub** على https://github.com (مجاني)
2. **أنشئ Repository جديد** (مستودع)
   - اسم: `masar-portal-app`
   - اجعله Private (خاص) إن أردت
3. **ارفع محتويات هذا المجلد** للمستودع:
   - إما عبر واجهة الويب (اسحب الملفات)
   - أو عبر Git command line
4. **انتظر 5 دقائق** — GitHub Actions ستبني التطبيق تلقائيًا
5. اذهب إلى تبويب **Actions** في مستودعك
6. اضغط على آخر تشغيل ناجح (أخضر ✓)
7. في الأسفل ستجد **Artifacts** → حمّل `MasarPortal-apk`
8. ستجد ملف `app-debug.apk` بداخله — هذا ملف التطبيق

### الخيار 2 — تجميع محلي (يحتاج Android Studio)

1. ثبّت Android Studio من https://developer.android.com/studio
2. افتح هذا المجلد في Android Studio
3. اختر Build → Build APK
4. سيظهر الملف في `app/build/outputs/apk/debug/`

## كيف تثبت APK على جوال المندوب

1. أرسل ملف `app-debug.apk` للمندوب (واتساب/تيلجرام/USB)
2. على جوال المندوب: افتح الملف
3. سيطلب الجوال السماح بـ "تثبيت من مصادر غير معروفة" — اضغط نعم
4. يثبّت ويفتح
5. **عند أول تشغيل** يطلب التطبيق رابط الموقع: `vr2.umniate.com`
6. بعدها يفتح بوابة المناديب مباشرة

## ضبط المناديب في النظام

قبل أن يستخدم المندوب التطبيق:

1. ادخل لوحة التحكم
2. **سجل المناديب** → اختر المندوب
3. في الأسفل ستجد قسم **«دخول تطبيق المندوب»**
4. اكتب كلمة مرور للمندوب
5. اضغط حفظ
6. أعطِ المندوب رقم إقامته + كلمة المرور — هذه بيانات دخوله للتطبيق

## تخصيص شعار التطبيق

من لوحة التحكم → **إدارة المستخدمين** → قسم العلامة التجارية:
- يمكنك رفع شعار جديد (يُحدَّث في الموقع + لوحة التحكم + الفواتير)
- تغيير اسم التطبيق
- تغيير اللون الأساسي

**ملاحظة:** أيقونة التطبيق المثبتة على جوال المندوب لا تتغير تلقائيًا — هي مدمجة في APK. لتغيير الأيقونة على الجوال يجب بناء APK جديد بشعار جديد.
